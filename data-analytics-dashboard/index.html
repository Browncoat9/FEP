📊 Data Analytics Dashboard

An AI-powered dashboard that surfaces insights from raw data and presents them in plain language.

Overview

This project combines traditional data visualization with AI-generated narrative summaries. Rather than requiring users to interpret charts themselves, the dashboard explains what the data means in plain English — highlighting trends, anomalies, and actionable takeaways.

Technologies

Python Pandas Chart.js Claude API HTML/CSS

Key Features

Upload any CSV and receive an instant AI-generated summary of key trends, outliers, and recommended next steps — alongside interactive charts.
